# Projekt-Anweisungen

- Nach jeder Änderung (Edit/Write an Dateien im Repo) automatisch `git add`, `git commit` und `git push` auf den aktuellen Branch ausführen.
- Commit-Message kurz und auf Deutsch, beschreibt die Änderung.
- Keine Rückfrage vor dem Push — nur auf dem aktuellen Branch (kein force-push, kein Branch-Wechsel).
