So einbinden: Auf „Signatur kopieren" klicken → im Mail-Programm die Signatur-Einstellungen öffnen → einfügen (⌘V / Strg+V).
  • Apple Mail: Einstellungen → Signaturen → neue Signatur, einfügen (Haken „Standardschrift verwenden" entfernen).
  • Gmail: Einstellungen → Allgemein → Signatur → einfügen.
  • Outlook: Datei → Optionen → E-Mail → Signaturen → einfügen.
